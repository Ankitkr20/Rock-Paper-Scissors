## 🛠️ Features
Interactive UI
Score tracking for user and computer
Random computer choices
Visual result feedback (win, lose, draw)

## 🧠 Logic
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
Draw if both choices are the same
